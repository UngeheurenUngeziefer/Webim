Технологии которые я использовал

CSS

CSS (англ. Cascading Style Sheets — каскадные таблицы стилей) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.

HTML

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный интерпретируемый браузерами язык разметки документов.

Django

Django (Джанго) — фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC. Проект поддерживается Django Software Foundation.

JavaScript

JavaScript (аббр. JS) — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили.

Python

Python (в русском языке распространено название питон) — высокоуровневый язык программирования общего назначения, ориентированный на повышение читаемости кода.

Adobe Photoshop

Adobe Photoshop — многофункциональный графический редактор, разработанный и распространяемый фирмой Adobe Systems. В основном работает с растровыми изображениями.
